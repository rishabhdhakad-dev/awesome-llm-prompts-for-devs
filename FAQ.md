# FAQ

## What is this repository for?

It is a curated collection of developer-focused prompts for code review,
debugging, architecture, testing, security, docs, DevOps, Git, API design, and
more.

## How many prompts are there right now?

The repository currently has about 95 prompts. The README now reflects that
more honest count.

## Can I use these prompts with any LLM?

Yes. The prompts are written to work with Claude, ChatGPT, Copilot, and other
general-purpose models.

## Why are placeholders used?

Placeholders like `[YOUR CODE]` and `[LANGUAGE]` make the prompts reusable
across different projects and stacks.

## How do I contribute?

See [CONTRIBUTING.md](CONTRIBUTING.md) for the template, structure, and PR
guidelines.

## Where should new prompts go?

Put them in the matching folder under `prompts/<category>/README.md`.
