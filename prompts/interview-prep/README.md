# Interview Prep Prompts

### System Design Mock Interview
**Use case:** Practice a realistic system design interview with feedback.
**Works with:** All LLMs

```text
Act as a senior staff engineer conducting a system design interview.

Interview me for:

[ROLE OR COMPANY TYPE]

Context:
- Experience level: [LEVEL]
- Time limit: [TIME]

Ask one question at a time, wait for my answer, then provide feedback and
follow-up questions.

Output format:
1. Interview question
2. Feedback on my answer
3. Stronger answer outline
4. Follow-up question
```

### Behavioral Interview Coach
**Use case:** Prepare concise answers to behavioral questions.
**Works with:** All LLMs

```text
Act as a senior engineering manager coaching me for interviews.

Help me answer this behavioral question:

[QUESTION]

Context:
- My experience: [EXPERIENCE]
- Role I’m applying for: [ROLE]

Create a strong STAR-style answer, then show a shorter version and note where I
should personalize it.

Output format:
1. STAR answer
2. Short version
3. Personalization notes
```

### DSA Practice Generator
**Use case:** Generate algorithm practice problems with hints.
**Works with:** All LLMs

```text
Act as an interview prep mentor.

Generate a data structures and algorithms practice session for:

[TOPIC]

Context:
- Difficulty: [EASY/MEDIUM/HARD]
- Language: [LANGUAGE]

Provide one problem, hints, an optimal approach, and complexity analysis without
giving away the solution immediately.

Output format:
1. Problem statement
2. Hints
3. Optimal approach
4. Complexity
5. Follow-up variant
```

### Resume Bullet Improver
**Use case:** Rewrite resume bullets for clarity and impact.
**Works with:** All LLMs

```text
Act as a senior recruiter and engineering manager.

Rewrite these resume bullets:

[YOUR BULLETS]

Context:
- Role target: [ROLE]
- Tech stack: [STACK]
- Impact style: [METRIC-HEAVY / CLEAR / EXECUTIVE]

Make them concise, quantified, and result-oriented.

Output format:
1. Improved bullets
2. Why they are stronger
```

### Interview Question Explainer
**Use case:** Understand what a question is really asking.
**Works with:** All LLMs

```text
Act as an interview coach.

Explain this interview question:

[QUESTION]

Context:
- Role: [ROLE]
- Level: [LEVEL]

Break down what the interviewer is testing, common mistakes, and a strong
approach to answering.

Output format:
1. What the question tests
2. Common mistakes
3. How to structure the answer
4. Example answer outline
```
