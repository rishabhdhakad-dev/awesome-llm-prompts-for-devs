# Roadmap

This roadmap tracks the next most valuable expansions for the repository.

## Short Term

- Expand each existing prompt pack to 10 to 15 prompts
- Add `frontend`, `mobile-development`, `system-design`, and `observability`
- Add a `CONTRIBUTORS.md` or `THANKS.md` page
- Add example screenshots or rendered prompt previews

## Medium Term

- Add tags to the prompt index for difficulty and use case
- Add machine-readable metadata for each prompt pack
- Add release notes and versioned changelog entries
- Add copy-ready snippets for VS Code

## Long Term

- Reach a truly accurate `500+` prompt collection
- Add multilingual prompt packs
- Add provider-specific variants for Claude, ChatGPT, and Copilot
- Add a lightweight website or docs site for browsing prompts
